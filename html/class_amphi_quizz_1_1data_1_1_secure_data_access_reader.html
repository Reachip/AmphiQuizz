Detailed Description

SecureDataAccessReader fournit une couche supplémentaire à DataAccess afin de fournir des méthodes qui assure la sécurité des transactions avec la base de données.

SqlDataReader AmphiQuizz.data.SecureDataAccessReader.GetSecureReaderFor ( string  table,
string  whereClause = null 
)

Récupère une instance de SqlDataReader permettant de lire les données du table passé en paramètre avec une expression dans la clause where de la requête SQL.

Cette méthode tente préalablement une ouverture sécurisé de la connection. Si cela n'est pas possible, la méthode retourne une exception de type OpenConnectionFailureException.

Parameters
tableTable dont on souhaite récupérer le lecteur
whereClauseClause where de la requête
Returns
Retourne un objet chargé de lire la requête SQL ou null si la requête est incorrecte
Exceptions
OpenConnectionFailureExceptionInvoqué si la connexion n'a pas pu s'ouvrir
InvalidOperationExceptionInvoqué si la requête SQL est incorrecte
